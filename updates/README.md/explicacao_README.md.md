# Explicação de README.md

## Objetivo da mudança
Traduzir a documentação principal do projeto para português do Brasil, facilitando o entendimento para usuários e colaboradores que falam PT-BR.

## O que mudou (resumo)
- Texto do README reescrito de inglês para português, preservando a estrutura original.
- Seções de funcionalidades, comandos do bot e passos de instalação ajustadas para o idioma português.
- Tabelas de comandos e exemplos de uso mantidos, alterando apenas o idioma da descrição.

## Motivação/Contexto
Aprimorar a acessibilidade da documentação para a base de usuários em português, sem alterar o comportamento do código ou a forma de configuração do projeto. A mudança é exclusivamente documental.
